How to use JAM Circle

About Card

How to create Card?

  1. This is simple.Double click on Board,or on Lane.

How to modify Card?

  1. If you move cursor on Card,then is shown on the Card.
  2. Click icon,then open Card Edit Dialog.

How to move Cards between Board and Board?

  1. Select Lane or Board and open context menu(Please right-click and so on).

  2. Choose " Open Card List",then Card List Dialog is opened.

  3. Double Click "Select" icon,then Board List Dialog is opened,you can choose target board,and you double click target board.
  4. Target Board is opened,and you can put the card in the Card List to the board using drag and drop.

About Lane

How to create Lane?

  1. This is simple,too! Double click "Create Lane" icon.

How to modify Lane?

  1. If you move cursor on Lane,then is shown on the Lane.
  2. Click icon,then open Lane Edit Dialog.

How to define action to the Lane?

  1. Please open Lane Edit Dialog,then you can modify the script at the dialog.
  2. You can choose script language "JavaScript" or "JRuby".
  3. (Sorry,now it is not supported to use rubygems.)

How to iconize and reopen Lane?

  1. Please double click iconized Lanes.

About Board

How to create board.

  1. Please open the menu '[File]-[New Wizard]-[New Board wizard]'.
    2. 1.

How to define action to the Board?

  1. Open Board List Dialog.(If you want to do that,Please double click on "Select Board" icon.)

  2. Choose the board to define script, and open context menu(Please right-click at the board.)

What Short Cut is supported?

  • Ctrl+A

    Select All Cards on the board

  • Ctrl+Z

    Undo

  • Ctrl+Y

    Redo

  • Ctrl+=

    Zoom-in

  • Ctrl+-

    Zoom-out

  • Ctrl+C,Ctrl+V,Ctrl+X

    Copy,and Paste,and Cut selected objects.

  • Delete

    Delete selected objects.
